# ğŸ“– HR DATA FEED <a name="about-project"></a>

**HR data feed** is a web service that facilitates the retrieval of files from an SFTP server, parses CSV/TSV files and utilizes API calls to manage user data. Noteworthy features include dynamic user creation for non-existing records and seamless data updates for existing users. The HR Data Feed project comes with its own API endpoints for configuring the SFTP server and handling user-related operations at the destination.

## ğŸ’» Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone git@github.com:Nelitaa/hr-data-feed.git
```

### Install

Install this project with:

```sh
  cd hr-data-feed
  bundle install
```

### Usage

To run the project, execute the following command:

```sh
  ruby app.rb
```

To run tests, run the following command:
